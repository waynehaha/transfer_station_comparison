# 中转站输出价格对比工具使用说明

## 怎么启动

在 macOS 上，双击项目里的：

```text
启动价格对比.command
```

页面会自动打开：

```text
http://127.0.0.1:4173/index.html
```

如果第一次双击被系统拦截，请右键这个文件，选择「打开」。

## 数据保存在哪里

所有渠道数据保存在项目里的：

```text
data/providers.json
```

只要这个文件还在，换浏览器也不会丢数据。

## 导入导出

页面「详细换算表」右上角有：

- 导出数据：把当前渠道数据导出成 JSON 文件，方便备份或发给别人。
- 导入数据：把导入文件合并到当前数据里。

导入数据的规则：

- 完全一样的数据会自动跳过，不会重复添加。
- 名字不同的数据会直接追加。
- 名字一样但价格、倍率、充值金额等内容不同，会自动改名为「原名称（导入版）」；如果已经有导入版，会继续使用「原名称（导入版 2）」这类名字。

## 给别人使用

把整个文件夹发给对方即可。对方解压后，双击「启动价格对比.command」启动。

## 注意事项

- 不要直接双击 `index.html` 使用；直接打开网页时，无法把数据保存回项目文件。
- 使用期间请保留启动器打开的终端窗口；关闭窗口会停止本地服务。
- 如果 4173 端口被占用，启动器会尝试切换旧服务；如仍失败，请重启电脑后再打开。
